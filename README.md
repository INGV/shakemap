# ShakeMap4

## Quickstart
### Build docker
```
$ git clone git@gitlab.rm.ingv.it:shakemap/shakemap4.git
$ cd shakemap4
$ docker build --tag shakemap4 .
```

### Run docker container
Run docker container from shakemap4 image:
```
$ docker run -it --rm shakemap4 bash
```
